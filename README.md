Airline Delay Analytics (Lufthansa Case Study)
![Dashboard](dashboard.png)

📌 Overview
This project analyzes Lufthansa flight delays and cancellations using Power BI.
It focuses on identifying operational inefficiencies, delay patterns, and business optimization opportunities.

🎯 Project Objectives
•	Analyze flight delays and cancellations
•	Identify peak vs off-peak operational pressure
•	Distinguish between true delay vs process delay
•	Detect operational bottlenecks
•	Build a decision-ready dashboard

📊 Key KPIs
•	Total Flights
•	Delayed Flights
•	Cancelled Flights
•	On-Time Flights
•	Total True Delay Minutes
•	Total Process Delay Minutes

📈 Key Insights
•	Around 70% flights operate on time
•	Delay rate is ~22%, while cancellations remain below 8%
•	Large gap between process delays and passenger delays
•	Peak operational pressure affects delay frequency
•	Sunday shows higher delays despite lower flight volume

🧠 Business Insights
•	Delays are systematic, not random
•	Wide-body aircraft have higher delay impact
•	Process inefficiencies drive operational delays
•	Improving scheduling and buffer time can reduce delays significantly

🛠 Tools & Technologies
•	Power BI
•	DAX
•	Data Modeling (Fact Table approach)
•	Excel / CSV Data

💼 Business Value
•	Helps airlines optimize scheduling
•	Reduces operational costs
•	Improves on-time performance
•	Enhances customer satisfaction
